<h3 id="ex00">–£–ø—Ä–∞–∂–Ω–µ–Ω–∏–µ 00: –ü–æ–∏—Å–∫ –≤–µ—â–µ–π</h3>

–ù–∞ –ø–µ—Ä–≤–æ–º —ç—Ç–∞–ø–µ –¥–∞–≤–∞–π—Ç–µ —Ä–µ–∞–ª–∏–∑—É–µ–º —É—Ç–∏–ª–∏—Ç—É, –ø–æ—Ö–æ–∂—É—é –Ω–∞ `find`, —Å –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ–º Go. –û–Ω–∞ –¥–æ–ª–∂–Ω–∞ –ø—Ä–∏–Ω–∏–º–∞—Ç—å –ø—É—Ç—å –∏ –Ω–∞–±–æ—Ä –∫–æ–º–∞–Ω–¥–Ω—ã—Ö –æ–ø—Ü–∏–π, —á—Ç–æ–±—ã –∏–º–µ—Ç—å –≤–æ–∑–º–æ–∂–Ω–æ—Å—Ç—å –Ω–∞—Ö–æ–¥–∏—Ç—å —Ä–∞–∑–ª–∏—á–Ω—ã–µ —Ç–∏–ø—ã –∑–∞–ø–∏—Å–µ–π. –ù–∞—Å –∏–Ω—Ç–µ—Ä–µ—Å—É—é—Ç —Ç—Ä–∏ —Ç–∏–ø–∞ –∑–∞–ø–∏—Å–µ–π: –∫–∞—Ç–∞–ª–æ–≥–∏, –æ–±—ã—á–Ω—ã–µ —Ñ–∞–π–ª—ã –∏ —Å–∏–º–≤–æ–ª–∏—á–µ—Å–∫–∏–µ —Å—Å—ã–ª–∫–∏. –¢–∞–∫ —á—Ç–æ –º—ã –¥–æ–ª–∂–Ω—ã –∏–º–µ—Ç—å –≤–æ–∑–º–æ–∂–Ω–æ—Å—Ç—å –∑–∞–ø—É—Å–∫–∞—Ç—å –Ω–∞—à—É –ø—Ä–æ–≥—Ä–∞–º–º—É —Å–ª–µ–¥—É—é—â–∏–º –æ–±—Ä–∞–∑–æ–º:

```

–∏–ª–∏ —É–∫–∞–∑–∞—Ç—å `-sl`, `-d` –∏–ª–∏ `-f`, —á—Ç–æ–±—ã –ø–µ—á–∞—Ç–∞—Ç—å —Ç–æ–ª—å–∫–æ —Å–∏–º–≤–æ–ª–∏—á–µ—Å–∫–∏–µ —Å—Å—ã–ª–∫–∏, —Ç–æ–ª—å–∫–æ –∫–∞—Ç–∞–ª–æ–≥–∏ –∏–ª–∏ —Ç–æ–ª—å–∫–æ —Ñ–∞–π–ª—ã. –û–±—Ä–∞—Ç–∏—Ç–µ –≤–Ω–∏–º–∞–Ω–∏–µ, —á—Ç–æ –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—å –¥–æ–ª–∂–µ–Ω –∏–º–µ—Ç—å –≤–æ–∑–º–æ–∂–Ω–æ—Å—Ç—å —è–≤–Ω–æ —É–∫–∞–∑–∞—Ç—å –æ–¥–Ω—É, –¥–≤–µ –∏–ª–∏ –≤—Å–µ —Ç—Ä–∏ –∏–∑ —ç—Ç–∏—Ö –æ–ø—Ü–∏–π, –Ω–∞–ø—Ä–∏–º–µ—Ä, `./myFind -f -sl /path/to/dir` –∏–ª–∏ `./myFind -d /path/to/other/dir`.

–í—ã —Ç–∞–∫–∂–µ –¥–æ–ª–∂–Ω—ã —Ä–µ–∞–ª–∏–∑–æ–≤–∞—Ç—å –¥—Ä—É–≥—É—é –æ–ø—Ü–∏—é ‚Äî `-ext` (—Ä–∞–±–æ—Ç–∞–µ—Ç –¢–û–õ–¨–ö–û –µ—Å–ª–∏ —É–∫–∞–∑–∞–Ω–∞ -f), —á—Ç–æ–±—ã –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª–∏ –º–æ–≥–ª–∏ –ø–µ—á–∞—Ç–∞—Ç—å —Ç–æ–ª—å–∫–æ —Ñ–∞–π–ª—ã —Å –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–Ω—ã–º —Ä–∞—Å—à–∏—Ä–µ–Ω–∏–µ–º. –†–∞—Å—à–∏—Ä–µ–Ω–∏–µ –≤ —ç—Ç–æ–π –∑–∞–¥–∞—á–µ –º–æ–∂–Ω–æ —Ä–∞—Å—Å–º–∞—Ç—Ä–∏–≤–∞—Ç—å –∫–∞–∫ –ø–æ—Å–ª–µ–¥–Ω—é—é —á–∞—Å—Ç—å –∏–º–µ–Ω–∏ —Ñ–∞–π–ª–∞, –∫–æ–≥–¥–∞ –º—ã —Ä–∞–∑–¥–µ–ª—è–µ–º –µ–≥–æ —Ç–æ—á–∫–æ–π. –¢–∞–∫ —á—Ç–æ,

```

~$ ./myFind -f -ext 'go' /go

–í–∞–º —Ç–∞–∫–∂–µ –Ω—É–∂–Ω–æ –±—É–¥–µ—Ç —Ä–∞–∑—Ä–µ—à–∞—Ç—å —Å–∏–º–≤–æ–ª–∏—á–µ—Å–∫–∏–µ —Å—Å—ã–ª–∫–∏. –¢–∞–∫ —á—Ç–æ –µ—Å–ª–∏ `/foo/bar/buzz` ‚Äî —ç—Ç–æ —Å–∏–º–≤–æ–ª–∏—á–µ—Å–∫–∞—è —Å—Å—ã–ª–∫–∞, —É–∫–∞–∑—ã–≤–∞—é—â–∞—è –Ω–∞ –¥—Ä—É–≥–æ–µ –º–µ—Å—Ç–æ –≤ FS, –Ω–∞–ø—Ä–∏–º–µ—Ä, `/foo/bar/baz`, –Ω–∞–ø–µ—á–∞—Ç–∞–π—Ç–µ –æ–±–∞ –ø—É—Ç–∏, —Ä–∞–∑–¥–µ–ª–µ–Ω–Ω—ã–µ `->`, –∫–∞–∫ –≤ –ø—Ä–∏–º–µ—Ä–µ –≤—ã—à–µ.

–ï—â–µ –æ–¥–Ω–∞ –≤–µ—â—å –æ —Å–∏–º–≤–æ–ª–∏—á–µ—Å–∫–∏—Ö —Å—Å—ã–ª–∫–∞—Ö ‚Äî –æ–Ω–∏ –º–æ–≥—É—Ç –±—ã—Ç—å —Å–ª–æ–º–∞–Ω–Ω—ã–º–∏ (—É–∫–∞–∑—ã–≤–∞—Ç—å –Ω–∞ –Ω–µ—Å—É—â–µ—Å—Ç–≤—É—é—â–∏–π —Ñ–∞–π–ª). –í —ç—Ç–æ–º —Å–ª—É—á–∞–µ –≤–∞—à –∫–æ–¥ –¥–æ–ª–∂–µ–Ω –ø–µ—á–∞—Ç–∞—Ç—å `[broken]` –≤–º–µ—Å—Ç–æ –ø—É—Ç–∏ –Ω–∞–∑–Ω–∞—á–µ–Ω–∏—è —Å–∏–º–≤–æ–ª–∏—á–µ—Å–∫–æ–π —Å—Å—ã–ª–∫–∏.

–§–∞–π–ª—ã –∏ –∫–∞—Ç–∞–ª–æ–≥–∏, –∫ –∫–æ—Ç–æ—Ä—ã–º —É —Ç–µ–∫—É—â–µ–≥–æ –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—è –Ω–µ—Ç –¥–æ—Å—Ç—É–ø–∞ (–æ—à–∏–±–∫–∏ —Ä–∞–∑—Ä–µ—à–µ–Ω–∏–π), –¥–æ–ª–∂–Ω—ã –±—ã—Ç—å –ø—Ä–æ–ø—É—â–µ–Ω—ã –≤ –≤—ã–≤–æ–¥–µ –∏ –Ω–µ –≤—ã–∑—ã–≤–∞—Ç—å –æ—à–∏–±–∫—É –≤—Ä–µ–º–µ–Ω–∏ –≤—ã–ø–æ–ª–Ω–µ–Ω–∏—è.
